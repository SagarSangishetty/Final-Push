Java --- pom.xml is used to download the dependency usin command "mvn dependency:go-offline" & mvn clean package (to build the code)

to build image : maven:3.9-eclipse-temurin-21
to run the code image : eclipse-temurin:21-jre

Once u build maven will create a jar file <dockerworkdir>/target/demo-0.0.1-SNAPSHOT.jar & you can run the jar file to host the application "java -jar <jarfile>"
You need to rename the jar file demo-0.0...(demo... is default name maven will give to jar file) to something short show in below command in run time stage
"COPY --from=builder /app/target/demo-0.0.1-SNAPSHOT.jar  app.jar"
