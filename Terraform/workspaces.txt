terraform workspaces -- 

A workspace allows you to maintain multiple state files for the same Terraform configuration.
They are commonly used to separate environments such as dev, qa, and prod while reusing the same code.


-- commands --

terraform show   ---   Show the name of the current workspace

terraform workspace new  <wrokapace name> ---  Create a new workspace

terraform workspace list   ---   List Workspaces

terraform workspace delete  ---  Delete a workspace

terraform workspace select  ---  Select a workspace
